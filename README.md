# AnalisePorPosicao-Quina

Projeto para análise estatística dos resultados da Quina por posição.

## Descrição

Este projeto analisa os resultados históricos da loteria Quina, focando na análise por posição dos números sorteados. A Quina sorteia 5 números de 1 a 80, e este projeto permite:

- Analisar a frequência de números em cada posição (1ª, 2ª, 3ª, 4ª e 5ª posição)
- Identificar padrões e tendências nos sorteios
- Gerar estatísticas úteis para análise
